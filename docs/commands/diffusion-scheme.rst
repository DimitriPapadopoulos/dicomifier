diffusion-scheme
================

Convert dMRI meta-data stored in JSON (aliases: *diffusion*, *diff*).

**Usage**::
  
  dicomifier diffusion-scheme [options] source format destination

Arguments
---------

.. option:: source
  
  Source JSON file

.. options:: format
  
  Output format, among:
  
  - *mrtrix*

.. option:: destination
  
  Output file

Options
-------

.. option:: -v <level>, --verbose <level>
  
  Verbosity level (*warning*, *info*, or *debug*)
